# OptiFlow

This project focusses mainly on solving the problem of excess commute time of daily travelling and also helps to minimize any room of error in road safety and Traffic management. 

-> This project helps reduce commute time based on congestion on each lane of an intersection.
-> It takes Direct input from the Camera or Video Feed to detect Vehicles and pedistrians and optimizes the Traffic Light Timing.
-> YOLO model is used for the detection of Vehicles.
-> Dynamic Green Light timing based on vehicle count per lane
